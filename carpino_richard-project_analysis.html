Project Analysis

Client: Spring Lake Elementary

Project: Website Redesign

Designer:Richard Carpino

Site Intent

Current Site:

The Spring Lake Elementary website currently features an unorganized theme that creates a less than desirable user experience. The navigation contains twenty-one links not including an unnecessary link for the current page right below that when clicked will just take you to the page you are already viewing. Tables and poor use of heading tags and non-semantically written html is the norm on every page it seems, the site does not validate correctly using the W3.org validator.

Project Goal:

The new Spring Lake Elementary website will feature a new streamlined theme with updated navigation menu. The current menu has many categories, most which can be combined into the same page, which with fewer categories will make the user experience less confusing and more enjoyable. Code will be semantically written with Web Standards in mind to ensure proper utilization of bandwidth so pages will load optimally.

Competitive Analysis

Overview:

The Spring Lake Elementary website resides on the k12.fl.us domain which houses not only all of the Seminole county school district schools but all of the other Florida school districts as well. The schools are not in competition with each other but each contains websites that are more useful than others.

Keywords:

Spring Lake Elementary, Spring Lake Elementary Florida, Seminole County Schools, Florida elementary schools, Seminole schools, Seminole Elementary, Spring Lake School.

Demographics:

The highest usage based on Alexa.com statistics where: Female between 35-44 years of age with some college education and children in the household. Average income is 60k-100k.

Web Site Statistics:

  • 35% of users come to the site from Google and 4% or less come from other search engines.
  • 7% of users on k12.fl.us domain go to the scps subdomain, which is Spring Lake Elementary school.

Targeted Audience

The targeted audiences are parents that have kids currently enrolled in Spring Lake Elementary or looking to enroll. Specifically the target audience will be that mentioned in the demographics section.
